# Pro Tic-Tac-Toe Web Application

A professional, modern, and responsive Tic-Tac-Toe web application built with vanilla HTML, CSS, and JavaScript. It features a stunning dark mode design and an unbeatable AI opponent powered by the Minimax algorithm.

## Features

- **Modern UI/UX**: clean dark theme with glassmorphism effects and smooth animations.
- **Game Modes**:
  - **PvP**: Classic Player vs Player on the same device.
  - **vs AI**: Challenge an "Unbeatable" AI powered by the Minimax algorithm.
- **Responsive Design**: Fully optimized for desktop and mobile devices.
- **Scoreboard**: Tracks wins and ties dynamically during the session.

## Technologies Used

- **HTML5**: Semantic structure.
- **CSS3**: Custom properties (variables), Flexbox, Grid, and Animations.
- **JavaScript**: ES6+, Minimax Algorithm for AI Logic.

## How to Run

1.  Clone the repository:
    ```bash
    git clone https://github.com/Fizzza777/PRODIGY_WD_03.git
    ```
2.  Navigate to the project directory:
    ```bash
    cd PRODIGY_WD_03
    ```
3.  Open `index.html` in your web browser.
